## 🔖 0. 소개

위 스터디 프로젝트는 네이버 쇼핑, 호텔스 컴바인과 같은 통합 비교사이트처럼 온라인 강의 정보를 한 곳에 모아 비교해주는 웹 서비스와 효율적인 학습을 위한 스터디 모집 서비스를 제공합니다.

- **프로젝트명** : WeStudy
- **개발 기간** : 2020.06.22 ~ 2020.07.26
- **인원** : 5명
- **담당 역할** : 백엔드
  - 프로젝트 개발 환경 구축
  - ERD 모델링
  - 쪽지 및 채팅 기능
  - 온라인 강의 정보 크롤링
  - 게시판 카테고리 필터링
  - 관리자 회원 및 신고관리 기능
  - 스터디 현황관리 기능
